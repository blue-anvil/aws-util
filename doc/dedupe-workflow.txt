Deduplicate staging bucket against prod
---------------------------------------

Set credentials via env vars before running (copy-paste per bucket as needed):

    export AWS_ACCESS_KEY_ID=...
    export AWS_SECRET_ACCESS_KEY=...
    export AWS_DEFAULT_REGION=...

Step 1 - Hash prod bucket:

    python generate_hashes.py -b my-prod-bucket -f prod.txt

Step 2 - Hash staging bucket:

    python generate_hashes.py -b my-staging-bucket -f staging.txt

Step 3 - Find files that exist in both with matching hash:

    python compare_hashes.py prod.txt staging.txt -f to_delete.txt

Step 4 - Verify what will be deleted:

    python delete_files.py -b my-staging-bucket -f to_delete.txt -t

Step 5 - Execute deletions:

    python delete_files.py -b my-staging-bucket -f to_delete.txt
